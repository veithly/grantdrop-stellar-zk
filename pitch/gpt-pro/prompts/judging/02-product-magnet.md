## Scoring

Score each shortlisted candidate from 0-5 on:

- First 5 seconds: judge immediately understands the human pressure or desire.
- First action: judge can personally do the first action in <=30 seconds.
- 5-second clip: the state change is visually replayable and video-friendly.
- Input mutation: judge changes one input and the result clearly changes.
- Desire/power pull: user gains money, access, status, reward, fairness, privacy, or competition upside.
- Retell line: a non-domain judge can repeat the scene in plain language.
- Shareable artifact: output is a receipt, pass, score card, settlement card, reward pass, or remittance slip.
- Anti-boring mutation: it avoids compliance form, admin console, proof log, and generic dashboard.
